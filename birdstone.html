About the Project

Birdstone is a passion project that showcases modern Android development practices. Built entirely in Kotlin, it leverages the power of Jetpack Compose for a beautiful, responsive UI and Firebase for seamless cloud synchronization.

The app is currently in private beta and access is granted on request. If you're interested in testing or collaborating, please reach out!

Tech Stack

Kotlin Jetpack Compose Firebase MVVM Coroutines Material 3

What makes Birdstone special

Architecture

Modern Architecture

Built with MVVM pattern, clean architecture principles, and unidirectional data flow for maintainable, testable code.

Cloud

Real-time Sync

Seamless cloud synchronization powered by Firebase, ensuring your data is always up-to-date across devices.
